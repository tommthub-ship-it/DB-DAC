## 취약점 신고

보안 취약점을 발견하셨다면 **공개 이슈**가 아닌 아래 방법으로 신고해 주세요.

- **이메일:** tommt.hub@gmail.com
- **제목 형식:** `[SECURITY] 간지-DAC - 취약점 설명`

신고 내용에 다음을 포함해 주세요:
1. 취약점 유형 (예: SQL Injection, 인증 우회 등)
2. 영향을 받는 컴포넌트 및 버전
3. 재현 방법 (PoC 코드 포함)
4. 예상 영향 범위

**48시간 이내** 초기 응답을 드립니다.

## 보안 정책

- 본 프로젝트는 대한민국 **개인정보보호법** 및 **ISMS-P** 준수를 목표로 합니다
- DB 자격증명은 절대 코드/로그에 평문으로 노출되지 않습니다
- 감사 로그는 무결성 보장을 위해 수정 불가능한 저장소에 보관을 권장합니다
